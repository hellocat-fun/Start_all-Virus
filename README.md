## 项目概述

本项目是一个批处理脚本集合，具有以下核心功能：

- **文件加密模块** (`jiami002020.bat`)：对users目录下的文件进行 AES+RSA 混合加密  
- **病毒演示模块** (`vis.bat`)：创建无线个 CMD 窗口并持续阻止用户结束任务，极具病毒感 
- **休息提醒模块** (`break.bat`)：强制霸屏，期间锁定屏幕，防止用户任何阻止活动
- **启动管理器** (`start_all.bat`)：统一启动所有模块，并添加开机自启  

⚠️ **本软件设计具有高度侵入性，请务必在隔离的测试环境中使用。**

## 功能详解

| 模块 | 功能描述 |
|------|----------|
| `start_all.bat` | 主启动入口，检查管理员权限，安装开机自启，并依次启动各子模块 |
| `jiami002020.bat` | 遍历 `C:\users` 目录下指定扩展名的文件，使用 AES-256 加密，RSA 公钥加密 AES 密钥，生成 `.enc` 加密文件并删除原文件 |
| `vis.bat` | 启动 20 个守护进程，持续生成全屏 CMD 窗口（标题 `GUARDIAN_CMD_TITLE`），每 0.5 秒生成一个，阻止 Alt+F4 等关闭操作 |
| `break.bat` | 显示全屏黑色背景的 "WAITTING" 提示，持续 20 秒，期间禁用 Alt+F4 和 ESC，阻止用户关闭 |

## 使用说明

### 环境要求
- Windows 7 / 8 / 10 / 11  
- PowerShell（系统自带）  
- 管理员权限

### 启动
-解压压缩文件
-启动根目录下的Start_all.bat即可

## 问题

### 为什么我没有正常启动
-检查启动脚本有没有授予管理员权限

### 加密后的文件如何恢复
-复制解密密钥（你应该知道是哪个吧）

-运行password.bat

-将解密密钥复制进去，回车

-将生成的解密密码复制进上一个终端，回车，等待解密
